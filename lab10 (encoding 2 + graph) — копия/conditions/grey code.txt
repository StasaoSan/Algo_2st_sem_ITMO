Двоичный код Грея
Двоичный код Грея - способ перечисления всех 2" битовых строк из п бит таким образом, что при следующем перечислении изменяется только один бит, само изменения детерминировано (каждый бит в последовательности определенным образом изменяется при переходе от одного числа к следующему) и регулярно (каждое число в последовательности имеет одинаковое количество единичных битов).
Например, двоичный код Грея для п = 4 выглядит так: 0000, 0001, 0011,
0010,0110,0111,0101,0100,1100,1101,1112,1110,1010
1011, 1001, 1000
Входные данные
На вход подается п - порядок кода грея.
Выходные данные
Необходимо вывести 2^п строк, описывающих код Грея для п бит.
